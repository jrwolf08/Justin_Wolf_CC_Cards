# Api BlackJack Test Suite
The test suite was created to utilize the cards api to play a game of blackjack: ([https://www.gamesforthebrain.com/game/checkers/](https://deckofcardsapi.com/)) 
The suite utilizes PactumJs for the REST tool, Mocha as the test runner, and Chai as the assertion sutie.  
The sutie was developed using Node version 18.16.1.

Clone the repo and follow the installation instructions to get started!

# Installation
To get started, install the dependencies using npm.  
```
npm install
```

# Running the tests
To run the test suite use the following command.
```
npm run test
```
